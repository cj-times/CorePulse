# CorePulse
A specialized tool that Has Useful information such as, Your CPU/Model,RAM/Size,GPU/Information.

> [!NOTE]
> This is Still in Beta-testing, You may incunter some Bugs.

> [!WARNING]
> **Permissions:** CorePulse requires standard user permissions. If you encounter "Access Denied" errors, ensure you are not running the script from a restricted system directory (like `C:\Program Files`).

## 🚀 Features

* **Real-Time Telemetry:** Monitor your CPU, RAM, and Disk usage at a glance with a clean, responsive terminal interface.
* **Modular Architecture:** Designed with separate script components, making it easy for you to customize or extend the tool.
* **Automatic Updates:** Stay up to date with the latest improvements without needing to manually re-download the project.
* **Lightweight Performance:** Built to be efficient, ensuring the monitor itself doesn't consume significant system resources.
* **Simple Setup:** Get started in minutes with straightforward installation and minimal dependencies.
